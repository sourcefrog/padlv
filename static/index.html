pad.♥ short Launchpad URLs

pad.lv is a site-specific redirection service for Launchpad.net, so you can easily create short URLs to use in irc, mail, microblogging, or grafitti.

If you have suggestions, file a bug or send a merge proposal.

For updates, follow pad.lv on Google+.

By Martin Pool.

Patterns

http://pad.lv/1 or http://pad.lv/bug=1
Any bug, by number.
http://pad.lv/~mbp
A person or a team.
http://pad.lv/~
Your own person page.
http://pad.lv/p/bzr
A project.
http://pad.lv/fb/pad.lv or http://pad.lv/fb/u/bzr
File a bug against a project, or an Ubuntu source package. NB, if the package is not found, make sure you're using the source package name, not the binary name. For example, python2.7-doc comes from the source package python2.7.
http://pad.lv/b/~mbp
List of related bugs for a person or team.
http://pad.lv/b/pad.lv, http://pad.lv/b/u/glibmm2.4
List of bugs in a project, or bugs in an Ubuntu source package.
http://pad.lv/bas/~mbp, /baf, /bre, /bsu
List of bugs that are assigned to, affect, were reported by, or are subscribed to by a person.
http://pad.lv/q/1
A user-support question.
http://pad.lv/mps/bzr or http://pad.lv/mps/u/bzr
Active merge proposals in a project, or an Ubuntu source package.
http://pad.lv/u/bzr or http://pad.lv/d/bzr
A source package in Ubuntu or Debian respectively. (It must be the source package, not the binary name.)
http://pad.lv/c/unity
List of branches for a project.
http://pad.lv/c/~mbp/pad.lv/trunk
A branch.
http://pad.lv/ppa/james-w or http://pad.lv/ppa/james-w/polkit
A PPA (either the default or a specifically named one.)
http://pad.lv/bld/2321035
A package build job
http://pad.lv/files/ensemble or http://pad.lv/files/~mbp/+junk/repro-742217
Bazaar branch files in the trunk of a project, or any other branch.
http://pad.lv/OOPS-12345678
Details of a Launchpad internal error report. (Only accessible to Canonical Launchpad team members, and not always reliable because of OpenID glitches.)
